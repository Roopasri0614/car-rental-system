# Car Rental System

A full-stack web application that allows users to browse, book, and manage rental cars efficiently. The system provides an intuitive interface for customers and administrative controls for managing vehicles, bookings, and users.

---

## Overview

The Car Rental System simplifies the process of renting vehicles by providing a centralized platform where users can:

- Search available cars  
- Book vehicles for specific dates  
- Track booking status  
- Manage reservations  

Administrators can manage cars, users, and bookings through the backend.

---

## Features

### User Features
- User registration and login  
- Browse available cars  
- Book cars for selected dates  
- View booking history  
- Cancel bookings  

### Admin Features
- Add, update, and delete cars  
- Manage bookings  
- Manage users  
- Monitor system activity
